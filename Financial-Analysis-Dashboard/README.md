# Financial Analysis Dashboard

## Project Overview

This project is an interactive financial dashboard built in Power BI to analyze transaction performance, customer behavior, and regional trends.

## Tools Used

- Power BI
- DAX
- Power Query
- Excel

## Key KPIs

- Total Amount
- Total Transactions
- Average Transaction Value
- Total Fees
- Total Tax

## Features

- Monthly Trend Analysis
- Customer Segment Analysis
- Transaction Status
- State Analysis
- Gender Analysis
- Dynamic Slicers
