# Maven Market Performance Dashboard

## Objective
Analyze revenue performance, profitability, transactions, returns, and regional business trends using interactive Power BI dashboards.

## Tools Used
- Power BI
- DAX
- Data Modeling
- Power Query

## KPIs
- Current Month Revenue
- Current Month Profit
- Current Month Transactions
- Profit Margin %
- Return Rate %

## Visuals
- KPI Cards
- Geographic Map
- Treemap
- Revenue Trend Analysis
- Gauge Chart
- Interactive Slicers

## Business Insights
- Regional transaction performance
- Revenue target tracking
- Profitability analysis
- Weekly revenue trends
